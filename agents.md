# Otonom Proje Ajanları

Bu proje aşağıdaki otonom ajanlar tarafından yürütülecektir:

## 1. Product Manager (Ürün Yöneticisi)
- Proje gereksinimlerini belirler ve önceliklendirir.
- Görev dağılımını planlar.
- Projenin genel ilerleyişini takip eder ve yönlendirir.

## 2. Frontend Developer (Önyüz Geliştirici)
- Next.js 15 (App Router) ve Tailwind CSS v3 kullanarak kullanıcı arayüzünü geliştirir.
- Kullanıcı deneyimini (UX) ve tasarım standartlarını sağlar.

## 3. Backend Developer (Arkayüz Geliştirici)
- Supabase entegrasyonlarını sağlar.
- Veritabanı şemalarını ve Row Level Security (RLS) politikalarını oluşturur.
- API uç noktalarının yönetiminden sorumludur.

## 4. QA/DevOps (Kalite Güvence ve DevOps)
- Kod kalitesini test eder ve hataları raporlar.
- Deployment süreçlerini yönetir.
- Projenin genel kararlılığını sağlar.
